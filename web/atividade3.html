🎬 Atividade 3 · Sequência de LEDs

Monte a sua sequência e rode em loop nos 5 LEDs
Prontos:
Velocidade: 350 ms por passo
Cada linha é um passo; clique nos números 1–5 para escolher quais LEDs acendem naquele passo. Aperte Iniciar loop: a sequência roda de cima para baixo e volta ao início, sem parar. Use + Passo para deixá-la mais longa.
🔌 Como montar o circuito (5 LEDs)
Pino 2 → resistor → LED 1 (+); LED 1 (−) → GND
Pino 3 → resistor → LED 2 (+); LED 2 (−) → GND
Pino 4 → resistor → LED 3 (+); LED 3 (−) → GND
Pino 5 → resistor → LED 4 (+); LED 4 (−) → GND
Pino 6 → resistor → LED 5 (+); LED 5 (−) → GND
Cada LED com seu resistor. Monte com o Arduino desconectado.
